VR 교실 스트리머 미션 컨트롤러

사용 방법

  1. 각 스트리머의 미션 상태를 체크박스로 관리할 수 있습니다.
  2. 미션 제목 옆 ✏️ 버튼을 클릭하면 미션 내용을 직접 수정할 수 있습니다.
  3. 프로필 이미지를 클릭하여 스트리머의 프로필 사진을 변경할 수 있습니다.
  4. 업데이트 버튼을 클릭하면 해당 스트리머의 미션 현황판이 실시간으로 업데이트됩니다.
  5. 스트리머는 OBS에 미션 현황판 링크를 브라우저 소스로 추가하면 됩니다.
